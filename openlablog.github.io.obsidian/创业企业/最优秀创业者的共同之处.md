“如果你只观察创业者采取的行动，你会得出结论说，研究他们不会给你带来多少收获。每个创业者的行为和他们本人一样具有特异性。你必须得是拉里·佩奇（Larry Page）和谢尔盖·布林（Sergey Brin）才能创立谷歌，你必须得是奥普拉·温弗瑞（Oprah Winfrey）才能创立哈普娱乐集团（Harpo Productions）。”

创业者的特点：成就、销售、坚毅、领导力、自省、人脉、处理失败、步步为营、客户服务

在我妻子筹备我家一年一度的假日派对之际，现在和年底之间的某个时刻，在布朗 - 戴维斯大宅将会发生以下事情。

我们的一位客人——我妻子似乎每年都要在她能邀请的客人数量上刷新纪录——会转向我，问道：“保罗，你在写什么书呢？”

我会尽我最快速度向他介绍我写的创业类书籍，这个与我谈话的人会非常礼貌地倾听，然后他们会说：

“你已经写了很长时间的有关创业者的书了，那么最优秀的创业者间有什么共同点吗？”

（他们会努力转变话题，没人想听一个作者介绍他正在写的东西，不论他介绍得有多快，即使这个话题是他们先问起的。）

过去，当人们想知道最优秀的创业者有哪些共同点时，我总会这么回答（我现在也是这么回答的）：

“如果你只观察创业者采取的行动，你会得出结论说，研究他们不会给你带来多少收获。每个创业者的行为和他们本人一样具有特异性。你必须得是拉里·佩奇（Larry Page）和谢尔盖·布林（Sergey Brin）才能创立谷歌，你必须得是奥普拉·温弗瑞（Oprah Winfrey）才能创立哈普娱乐集团（Harpo Productions）。”

但是——这是一个很大的但是——如果你看看他们的思考方式，你会看到明显的相似性。他们在创立自己的公司时，几乎都遵循了这样的过程。他们：

1.弄清楚他们真正想做的是什么；让他们迸发激情的是什么。换句话说，他们牢牢把握着他们的目标。

2. 向那个目标迈出一小步。
3. 在迈出那一小步后停下来，看看他们学到了什么。
4. 对学到的知识举一反三并再迈出一小步。
5. 迈出那一步后停顿一下。
6. 对第二步中学到的知识举一反三，然后再迈出一小步……

如果我们要将这一过程简化为一个公式，那就是行动。学习。举一反三。如此往复。

换句话说，他们花很多时间进行策划，或玩“假如……会怎样”的游戏。你永远不会真正了解万事万物会如何反应，直到你给了它某样东西让它有所反应。”

这种说法是对的。但通常问这个问题的人并不想知道这么多，尤其是如果他们了解到横亘在他们和我妻子著名的巧克力山核桃馅饼之间的唯一阻隔就是我喋喋不休地说着我的文章，“行动。学习。举一反三。重复模式”（Act. Learn. Build. Repeat model）。

所以，今年如果有人问我创业者取得成功的秘诀，我将把我的回答浓缩为以下 26 个字：

创业者热衷于发现机遇。一旦他们发现了，就会立刻行动。

当然，这个回答有一部分是不言而喻的。创业者真干，实干。（如果你只是坐在那儿设想新产品或服务，而不采取任何行动，你就不是一个创业者。你只是一个有着很多没有实现的想法的人。）

如果有人仍对成功的创业者如何行事感到好奇，我会向他们详细介绍“行动。学习。举一反三。重复模式”。

但我认为，这 26 个字抓住了创业者的两个关键特点。创业者总在寻找他们能够满足的市场需求，一旦他们找到了，他们会干劲十足地开展工作，看看他们是否能提出一个解决方案。这种干劲让他们迈出一小步又一小步，以检查自己是否正确。

如果没什么别的问题了，我的回答会让大家加快步伐奔向山核桃馅饼。
